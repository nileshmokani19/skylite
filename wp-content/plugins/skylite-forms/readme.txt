Responsive Form is a simple WordPress plugin that generates a responsive form on your website or blog.

== Description ==


Features:

* Easy to install.
* It is responsive form.
* Easy jQuery validations.
* Easy to understand coding standard.
* Manage form style/CSS from admin pannel.



== Installation ==

1. Upload the `skylite-forms` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin using the 'Plugins' menu in your WordPress admin panel.
3. You can adjust the necessary settings using your WordPress admin panel in "Settings" "Responsive Contact Form Settings".
